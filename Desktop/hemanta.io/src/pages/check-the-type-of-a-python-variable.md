We can check the type of a Python variable using the ~~type()~~ function.

```py {numberLines}
integer = 24

float = 12.24

name = "Hemanta"

print(type(integer))
# <class 'int'>

print(type(float))
# <class 'float'>

print(type(name))
# <class 'str'>
```
