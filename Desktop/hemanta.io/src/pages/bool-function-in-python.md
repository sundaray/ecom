In Python, we can use the ~~bool()~~ function to convert any data type to a boolean (~~True~~ or ~~False~~).

**Example:**

```py {numberLines}
print(bool(0))
# False

print(bool(1))
# True

print(bool(True))
#True

print(bool(False))
#False
```
